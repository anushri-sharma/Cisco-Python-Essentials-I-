# 3.4.6   LAB   The basics of lists
## Scenario
There once was a hat. The hat contained no rabbit, but a list of five numbers: 1, 2, 3, 4, and 5.

Your task is to:

1. write a line of code that prompts the user to replace the middle number in the list with an integer number entered by the user (Step 1)
2. write a line of code that removes the last element from the list (Step 2)
3. write a line of code that prints the length of the existing list (Step 3).

```python
hat_list = [1, 2, 3, 4, 5]  # This is an existing list of numbers hidden in the hat.

# Step 1: write a line of code that prompts the user
# to replace the middle number with an integer number entered by the user.

input_number = int(input("Enter the number: "))
hat_list[2] = input_number
print("New List Content: ", hat_list)

# Step 2: write a line of code that removes the last element from the list.
del hat_list[4] 

print("Deleted List content", hat_list)
# Step 3: write a line of code that prints the length of the existing list.

print("List after deleted length", len(hat_list))
print(hat_list)


```

```output

Enter the number: 21
New List Content:  [1, 2, 21, 4, 5]
Deleted List content [1, 2, 21, 4]
List after deleted length 4
[1, 2, 21, 4]

```
